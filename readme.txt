This plugin enables left and right key post navigation. 

== Description ==

This plugin gives you an option to goto next post by pressing right key twice. Pressing Right arrow key once gives hint on what post is next. Same applies to Left arrow key. When its clicked once a simple title for next post is shown, Pressed twice takes you to previous post.

== Installation ==

1. Upload unzipped plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Plugin Options on left menu under Settings > JCWP Left Right key.
